Work

Projects

Here are a few cool projects I've worked on, feel free to check out the code on Github

Firehive

As part of Israel Tech Challenge's annual hackathon, I was part of a team that built a proof of concept of a tool to be used as a firewall for IOT devices. We wrote scripts in Python to generate simulated network capture data and stored the data using AWS Elasticsearch. We demonstrated our project by developing a web app using Python Flask, Kibana for data visualization, and hosted the web app on Heroku.

Our team was awarded second place by a panel of judges who evaluated projects based on the sophistication of the technology used, the delivery of our pitch, and the long-term plan for the project.

WhatStats

I built a small web app using Python Flask to calculate chat statistics for Whatsapp chat data. Visit the site to see it live.
