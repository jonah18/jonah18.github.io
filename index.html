Google

This past summer I interned for Google on the DoubleClick team in Kirkland, WA. My project was to design, implement, and test a feature that allows advertisers to see graphs describing how the size of their audiences' sizes have changed over time.

I wrote server side code in Java, using tools like Google Guice, Promise Graphs, and Java Streams. The client side was implemented in Dart.

Objective Arts

I worked as a Software Engineer for Objective Arts on a tool to detect and extract specific data in identification forms for their Medicaid Eligibility platform. I'm making use of several APIs, including Google Cloud Vision, and services such as AWS Lambda and S3.

Loom Systems

I also developed notification integrations with services like Slack, PagerDuty, OpsGenie, and AWS SNS.

LessonSketch

Firehive

As part of Israel Tech Challenge's annual hackathon, I worked on a team that built a proof of concept of a tool to be used as a firewall for IOT devices. We wrote scripts in Python to generate simulated network capture data and stored the data using AWS Elasticsearch. We demonstrated our project by developing a web app using Python Flask, Kibana for data visualization, and hosted the web app on Heroku.

Our team was awarded second place by a panel of judges who evaluated projects based on the sophistication of the technology used, the delivery of our pitch, and the long-term plan for the project.
